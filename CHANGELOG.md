# Changelog

All notable changes to Display Presets will be documented in this file.

The format follows Keep a Changelog loosely, and the project uses semantic
versioning while the public API is small.

## 0.1.0 - 2026-05-23

### Added

- Menu bar app for saving and applying display presets.
- Configuration window for creating and deleting presets.
- Optional Open at Login support through `SMAppService`.
- Install-in-Applications handoff for users running a temporary build copy.
- Local preset migration from the private `Display Mode Switch` app name.
- Future signed-release packaging targets.
- Runtime dependency check for `displayplacer`.
- Bundled app icon and icon-generation script.
- Project documentation for architecture, troubleshooting, releases, security,
  and contributions.
